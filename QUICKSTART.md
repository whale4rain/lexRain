# 快速开始

## 安装

```bash
# 1. 下载 ECDICT (https://github.com/skywind3000/ECDICT/releases/tag/1.0.28)
# 下载 ecdict-sqlite-28.zip (~700MB)

# 2. 解压到项目根目录
# 确保路径: ecdict-sqlite-28/stardict.db

# 3. 编译运行
cargo run --release
```

## 基本操作

### Dashboard 主界面
- `n` - 学习新词（自动选 20 个高质量词汇）
- `r` - 开始复习（今日到期的单词）
- `d` - 词典搜索
- `h` - 复习历史
- `s` - 学习统计
- `q` - 退出

### Review 复习模式
1. 看到单词后按 `Space` 显示答案
2. 左侧显示释义，右侧显示词形变化
3. `j/k` 滚动当前面板，`h/l/Tab` 切换面板
4. 按 `1-4` 评分：
   - `4` - 简单（瞬间回忆）
   - `3` - 良好（稍有思考）
   - `2` - 困难（勉强想起）
   - `1` - 忘记（完全不记得）

### Dictionary 词典
- 输入字母开始搜索
- `j/k` 或 `↑/↓` 浏览单词列表
- `h/l` 或 `←/→` 滚动详情面板
- `g` 跳到第一个，`G` 跳到最后一个

## 学习建议

### 首次使用
1. 按 `n` 学习 20 个新词
2. 全部过一遍（建议评分 2-3）
3. 第二天会出现复习提醒

### 每日流程
1. 打开应用，查看今日到期数量
2. 按 `r` 完成所有到期复习
3. 按 `n` 学习新词（推荐每天 10-20 个）
4. 使用 `d` 查询不熟悉的单词

### 评分标准
- **4 分太多** → 单词太简单，可以增加新词难度
- **1 分太多** → 学习量过大，减少每日新词数
- **2-3 分为主** → 正常学习状态

## ECDICT 特性

- **340万+ 词条** - 覆盖各类词汇
- **柯林斯星级** - ★★★★★ 表示重要度
- **牛津3000** - 核心词汇标记
- **考试标签** - 中考/高考/CET-4/CET-6/考研/TOEFL/IELTS/GRE
- **词形变化** - 过去式/分词/比较级/最高级等
- **词频数据** - BNC + 当代语料库

## 数据备份

学习进度保存在 `lexrain_progress.db`，定期备份此文件即可。
